# CodeCrypt Frontend

The live resurrection dashboard for CodeCrypt, featuring real-time metrics, 3D visualization, AI narration, and the Resurrection Symphony.

## Tech Stack

- **React 18** with TypeScript
- **Vite** for fast development and optimized builds
- **CSS Modules** with gothic/spooky theme
- **Server-Sent Events (SSE)** for real-time updates

## Project Structure

```
src/
├── components/     # React components
├── hooks/          # Custom React hooks (useEventSource, etc.)
├── utils/          # Utility functions
├── styles/         # CSS modules and global styles
├── types/          # TypeScript type definitions
├── App.tsx         # Main application component
└── main.tsx        # Application entry point
```

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Features

- **Live Metrics Dashboard**: Real-time charts showing resurrection progress
- **AI Narrator**: Audio narration of key events
- **3D Ghost Tour**: Interactive 3D visualization of codebase evolution
- **Resurrection Symphony**: Musical representation of code quality metrics

## Gothic Theme

The UI embraces a dark, gothic aesthetic with:
- Deep blacks and purples
- Glowing accents and shadows
- Smooth animations
- Resurrection-themed iconography
